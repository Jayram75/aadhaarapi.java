# Aadhaar API for Java
Aadhaar API for Java is a client library that provides an easy way to interact with the UIDAI Aadhaar services.

## Milestones
- [x] Authentication API 2.0
- [ ] OTP API 2.0
- [ ] e-KYC API 2.0
- [ ] Capture API 1.0
- [ ] Device Registration API 1.0
